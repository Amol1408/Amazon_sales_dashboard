# Amazon Sales Analytics Dashboard

## 📊 Project Overview
This project is an end-to-end data analytics solution built using SQL and Power BI. It analyzes e-commerce sales data to generate insights on revenue, customer behavior, returns, and delivery performance.

## ⚙️ Tools Used
- MySQL
- Power BI
- DAX
- ODBC Connectivity

## 📈 Key Insights
- Identified revenue trends and top-performing categories
- Analyzed customer behavior and segmentation
- Highlighted high return-risk products
- Evaluated delivery performance (On-time vs Late)


## 🚀 Key Features
- KPI Tracking (Revenue, Orders, Customers, AOV)
- Customer Segmentation (New vs Repeat)
- Return Risk Analysis
- Delivery Performance Monitoring

## 📌 Conclusion
This dashboard helps businesses make data-driven decisions by providing actionable insights into sales, customer behavior, and operational efficiency.
